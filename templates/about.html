About nScanner

nScanner is an information security tool for scanning, reconnaissance, and passive intelligence gathering. This project was developed by Anushree Balaji as part of a cybersecurity initiative.

Features

  • Port Scanning
  • Passive Recon using IPWhois
  • Flask Web Interface
  • Exportable Reports and Visualizations
