📘 About nScanner

Back to Home

nScanner is a network security analysis tool designed for educational and research purposes. It combines TCP scanning and passive reconnaissance with a clean web-based interface using Flask and Jinja2. The project helps you learn how network discovery tools work under the hood.

🚀 Features

  • TCP Port Scanner using python-nmap
  • Passive recon using IPWhois API
  • Matplotlib-based histogram for course data (if applicable)
  • Flask & Jinja2-based dynamic HTML rendering
  • Simple and lightweight UI
